# Real-Time Weather Dashboard

## Overview
This project is a responsive Weather Dashboard web application built using HTML, Bootstrap, JavaScript, and the OpenWeather API. It provides real-time weather information for any city entered by the user.

## Features
- Real-time weather data using OpenWeather API
- Search weather by city name
- Displays temperature, weather description, humidity, and wind speed
- Weather condition-based icons (Clouds, Rain, Clear, Snow, Mist)
- Search history with clickable previous cities
- Dark mode / Light mode toggle
- Input validation for correct city names
- Loading and error handling alerts
- Responsive and modern UI using Bootstrap

## Technologies Used
- HTML5
- CSS3 (Bootstrap 5)
- JavaScript (ES6)
- jQuery
- OpenWeather API

## How It Works
1. User enters a city name in the search bar.
2. The app sends a request to the OpenWeather API.
3. Weather data is fetched and displayed on the dashboard.
4. Search history is updated for quick access.
5. UI updates dynamically based on weather conditions.
